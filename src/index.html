@PROJECT:projectid=automation,,projecttitle=Automation,,projectimage=automation.jpg,,projecttitlecaps=AUTOMATION,,projecturl=https://github.com/jextrevor/Automation,,projectdesc=

Automation is a automatic music player and alarm clock for the Raspberry Pi. Capabilities include the following:

  • Automatic starting of music when movement in the room is detected (using the Raspberry Pi camera)
  • Automatic adjustment to slight changes in environment (gradual changing of light due to the sun, camera noise)
  • Music alarm at a certain time, time is changeable through the web interface
  • Notification LED when the program is "talking" (using the Raspberry Pi Camera LED)
  • Statement of the weather forecast for the day when the alarm is played (cloudy, partly cloudy, sunny, etc.)
  • Statement of how many unread emails the user has when music starts
  • User-settable reminders
  • Web interface to toggle alarm status, play or pause music, upload new music, and set reminders

The project is written in Python and is designed to run on the Model B. To run on newer models such as the B+ or Pi 2 you must change the CAMLED variable in the code to the GPIO port associated with the camera LED (32 for B+ or 2, 40 for Pi Zero).
